# fullstack-venturesity-jan25

Usage directions-

1. Start mongo instance
	mongod --dbpath <project path>\server\data\db

2. Start server
        node <project path>\server\index.js

3. Start client (type in browser)
        http://localhost:3000
		

Features-

1. Register new patients.
2. Manage patients - view all patient records.
